Luna

Luna is a personal life tracker inspired by various habit and mood trackers, all combined into a cohesive, easy-to-use platform. The goal of Luna is to provide users more flexibility with personalization while maintaining a clean and intuitive user-interface. Luna uses react with redux on the front-end to create an interactive and responsive web application. On the backend, this application takes advantage of the non-blocking architecture of express.js and the middleware and validation features provided by sequelize.

Artisan Alley

Artisan Alley is a e-commerce application modeled after Etsy created by myself and three other team members. This project uses flask for the backend and react for the frontend. Technologies used also includes: Python, JavaScript, CSS, HTML, SQLAlchemy, and Redux. Functionality includes three full CRUD features for products, reviews, and cart. As well as one partial CRUD for favorites.

Vacay Stays

Vacay Stays is a web application based on AirBnB, and also my very first full stack project! This was a solo project that had react frontend and express.js backend. Other technologies used in this project include: Javascript, Node.js, CSS, CSSGrid, HTML, SQLite3, PostgreSQL and Redux.
